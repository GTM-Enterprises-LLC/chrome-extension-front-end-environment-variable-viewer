React Environment Viewer Test Page

Use this page to test the Chrome extension

How to Test:

  1. Make sure the React Environment Viewer extension is installed
  2. Click the extension icon in your Chrome toolbar
  3. You should see the environment variables listed below
  4. Try the search and filter features
  5. Test the Copy and Export buttons
