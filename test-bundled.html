Bundled React App Simulation

This page simulates how environment variables appear in a real production React build.

⚠️ Important: This page contains minified/bundled JavaScript code similar to what Create React App produces. The extension should now be able to detect environment variables from this bundled code.

Simulated Bundle Patterns:

Example 1: Webpack DefinePlugin pattern
  e.REACT_APP_API_URL="https://api.production.com"

Example 2: Object literal pattern
  {REACT_APP_AUTH_DOMAIN:"auth.production.com"}

Example 3: Minified pattern
  "REACT_APP_VERSION":"2.5.1"

Example 4: NODE_ENV inline
  "production"
